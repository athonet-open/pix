      Pix.UserSettings (pix v0.0.0)

      Pix user settings.

      You can set some global settings in your ~/.config/pix/settings.exs file.

      The file should evaluate to a map conform to t/0.

      For example:

      %{
  env: %{
    "PIX_DEBUG" => "true"
  },
  command: %{
    run: %{
      cli_opts: [
        ssh: true
      ]
    },
    shell: %{
      cli_opts: [
        ssh: true
      ]
    }
  }
}
