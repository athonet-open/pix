    Pix.Config (pix v0.0.0)

    Types used across Pix project and pipeline configuration.

    The main user-facing type is pix_exs/0 — the shape returned by Pix.Project.project/0 in your .pix.exs file.

    Summary

    Types

    arch()

    Target CPU architecture or :noarch.

    args()

    Pipeline build arguments as key-value pairs.

    from()

    Pipeline source — either a local path or a git repository.

    from_git()

    Git-hosted pipeline source. :ref selects a branch/tag/commit, :sub_dir narrows to a subdirectory.

    from_path()

    Local filesystem pipeline source. :sub_dir narrows to a subdirectory containing pipeline.exs.

    pipeline_alias()

    User-chosen name that identifies a pipeline within a project.

    pix_exs()

    Raw project configuration as declared in .pix.exs.

    pix_exs_pipeline()

    Raw pipeline entry as declared in .pix.exs before resolution.

    supported_arch()

    CPU architectures supported by Pix.

    Types

    arch()

    @type arch() :: :noarch | supported_arch()

    Target CPU architecture or :noarch.

    args()

    @type args() :: %{required(String.t()) => String.t()}

    Pipeline build arguments as key-value pairs.

    from()

    @type from() :: from_path() | from_git()

    Pipeline source — either a local path or a git repository.

    from_git()

    @type from_git() :: %{
  :git => String.t(),
  optional(:ref) => String.t(),
  optional(:sub_dir) => Path.t()
}

    Git-hosted pipeline source. :ref selects a branch/tag/commit, :sub_dir narrows to a subdirectory.

    from_path()

    @type from_path() :: %{:path => Path.t(), optional(:sub_dir) => Path.t()}

    Local filesystem pipeline source. :sub_dir narrows to a subdirectory containing pipeline.exs.

    pipeline_alias()

    @type pipeline_alias() :: String.t()

    User-chosen name that identifies a pipeline within a project.

    pix_exs()

    @type pix_exs() :: %{pipelines: %{required(pipeline_alias()) => pix_exs_pipeline()}}

    Raw project configuration as declared in .pix.exs.

    pix_exs_pipeline()

    @type pix_exs_pipeline() :: %{
  from: from_path() | from_git(),
  default_args: args(),
  default_targets: [String.t()]
}

    Raw pipeline entry as declared in .pix.exs before resolution.

    supported_arch()

    @type supported_arch() :: :amd64 | :arm64

    CPU architectures supported by Pix.
