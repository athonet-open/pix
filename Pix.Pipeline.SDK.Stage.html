    Pix.Pipeline.SDK.Stage (pix v0.0.0)

    Represents a single stage in a Pix.Pipeline.SDK pipeline.

    Each stage corresponds to a FROM … AS <name> block in the generated Dockerfile. Stages carry their own instructions, build arguments, and declared output artifacts.

    Key properties:

    • :private — when true the stage is an internal build dependency and won't appear as a selectable target in pix run or pix ls.
    • :cache — when false the stage is rebuilt from scratch on every run (equivalent to --no-cache-filter).
    • :outputs — file paths declared via Pix.Pipeline.SDK.output/2 that are exported when running with --output.
